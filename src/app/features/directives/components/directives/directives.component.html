Directive PositiveNumber

Esta directiva se asegura de que los valores ingresados son un número positivo, no permite que escribas valores negativos.

Esta directiva se encarga de mostrar u ocultar un mensaje, se maneja mediante un botón, que se encarga de manejar por Input la visibilidad de la directiva.


Este es un mensaje que se puede mostrar u ocultar.
