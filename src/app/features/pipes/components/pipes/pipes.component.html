PIPES PERSONALIZADAS

Pipe filterList

Esta es una pipe que se encarga de filtrar por los nombres de una lista

  • {{ item.name }} - {{ item.breed }}

Pipe priceFormat

Agrega el formato de algún precio, en caso de que no se especifique en el html, se pondrá el valor por default que es el simbolo del euro.

{{ 1234.56 | priceFormat:'$' }}

{{ 1234.56 | priceFormat }}

Pipe sort

Devuelve en orden alfabético el listado. En este caso por la propiedad del array .name

  • {{ item.name }}

Pipe sort

Pipes angular

Devuelve en orden alfabético el listado. En este caso por la propiedad del array .breed

  • {{ 8736.3982 | number:'1.0-0' }}
  • {{ -3.6 | number:'1.0-0' }}
  • {{ 'LOWERCASE' | lowercase }}
  • {{ 'uppercase' | uppercase }}
  • {{ 'titlecase' | titlecase }}
  • {{ 3445 | currency:'USD' }}
  • {{ 3445 | currency:'EUR' }}
  • {{ '2419' | date:'shortDate' }}
  • {{ '2419' | date:'fullDate' }}
  • {{ 0.80 | percent }}
  • {{ 'Hello World' | slice:0:5 }}

Pipe Angular keyValue

Se usa para objetos clave valor, es muy útil sino sabemos o tenemos muchos elementos y no tenemos que apuntar las claves.

  • {{ item.key }}: {{ item.value }}
